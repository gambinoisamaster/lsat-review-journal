# LSAT Review Journal

A single-file web prototype for reviewing missed LSAT Logical Reasoning questions.
Log a question, tag its type / stimulus / difficulty / time, and capture your
**Reason Missed → Right Track → Takeaway** so mistakes get ingrained on review.

Built with a retro FIFA / Total Club Manager (PS2) aesthetic — vanilla HTML, CSS,
and JavaScript in one file. No build step.

## Live demo

Once GitHub Pages is enabled, the app will be live at:

`https://<your-username>.github.io/lsat-review-journal/`

## Run it locally

Download `index.html` and double-click to open it in any browser
(an internet connection is needed so the fonts load).

## Status

Early prototype. Filters, bookmarks, the Reviewed toggle, and the "Add New Question"
flow all work. Entries are held in memory for now, so they reset on refresh —
saving/persistence is a planned next step.
